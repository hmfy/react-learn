# react-learn

使用 Parcel 搭建的最小可运行 React 项目。

## 当前目录

```text
src/
  main.jsx
  App.jsx
  components/
    PageTitle.jsx
  pages/
    HomePage.jsx
  hooks/
  styles/
    global.css
```

其中：

- `pages/` 放页面组件
- `components/` 放可复用组件
- `hooks/` 预留给自定义 Hooks
- `styles/` 放全局样式

## 启动

```bash
npm install
npm run dev
```

开发服务默认会输出本地访问地址。

## 构建

```bash
npm run build
```
